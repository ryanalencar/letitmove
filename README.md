## About the project
It aims at studying and developing an application for listing movies according to gender in ReactJS with Typescript.

## Technologies
- ReactJs
- Typescript
- CSS | Styled Components
- NextJs

## Install

```sh
# Install the dependencies
yarn
```

## Usage

```sh
# Start the application
yarn dev
```
